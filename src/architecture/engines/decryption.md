# Decryption
The _decryption engine_ holds sufficient information to decrypt data encrypted to identities controlled by this node.

- _Inputs_
 Decryption instructions
- _Outputs_
 Decrypted data
- _Preferences_
 What to decrypt
- _Accounting_
 What was decrypted
